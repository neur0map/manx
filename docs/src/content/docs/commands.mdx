## Command Overview

```bash
manx [OPTIONS] <LIBRARY> [QUERY]
manx <COMMAND> [OPTIONS]
```

## Basic Commands

### Search Commands

#### `manx <library>`
Search for documentation of a specific library.

```bash
manx fastapi
manx react
manx django
```

#### `manx <library> <query>`
Search within a library's documentation.

```bash
manx fastapi middleware
manx react hooks
manx django models
```

#### `manx <library>@<version>`
Search version-specific documentation.

```bash
manx react@18
manx vue@3
manx angular@15
```

### Documentation Command

#### `manx doc <library> <query>`
Get comprehensive documentation instead of search results.

```bash
manx doc fastapi authentication
manx doc react useEffect
manx doc django orm
```

## Global Options

### `--limit <number>`
Limit the number of results displayed.

```bash
manx fastapi --limit 5      # Show only 5 results
manx react --limit 0        # Show all results (unlimited)
manx django --limit 20      # Show 20 results
```

Default: 10 results

### `--save <indices>`
Save specific search results to a file.

```bash
manx fastapi --save 1,3,5           # Save results 1, 3, and 5
manx react hooks --save 1-5         # Save results 1 through 5
manx django --save-all              # Save all results
```

### `--json`
Output results in JSON format (useful for scripting).

```bash
manx fastapi --json
manx react --save 1,2,3 --json      # Save as JSON
```

### `--offline`
Use cached results only (no network requests).

```bash
manx fastapi --offline
manx react hooks --offline
```

### `--quiet`
Minimal output mode (great for scripts).

```bash
manx fastapi --quiet | jq '.results[0].content'
```

### `--debug`
Enable debug logging for troubleshooting.

```bash
manx fastapi --debug
MANX_DEBUG=1 manx react  # Alternative method
```

### `--clear-cache`
Clear cache before searching.

```bash
manx --clear-cache fastapi
```

## Configuration Commands

### `manx config`
Manage Manx configuration.

#### Show current configuration
```bash
manx config --show
```

#### Set API key
```bash
manx config --api-key sk-your-context7-key
```

#### Set cache directory
```bash
manx config --cache-dir ~/my-cache
```

#### Enable/disable auto-caching
```bash
manx config --auto-cache on
manx config --auto-cache off
```

## Cache Commands

### `manx cache`
Manage the local cache.

#### View cache statistics
```bash
manx cache stats
```

Output:
```
Cache Statistics:
  Location: /home/user/.cache/manx
  Size: 12.3 MB
  Entries: 47
  Oldest: 2024-01-15
  Newest: 2024-01-20
```

#### List cached libraries
```bash
manx cache list
```

#### Clear entire cache
```bash
manx cache clear
```

#### Clear specific library cache
```bash
manx cache clear fastapi
```

## Export Options

### Save to Markdown
```bash
manx fastapi --save 1,2,3
# Creates: manx_results_[timestamp].md
```

### Save to JSON
```bash
manx fastapi --save 1,2,3 --json
# Creates: manx_results_[timestamp].json
```

### Save all results
```bash
manx fastapi --save-all
```

### Specify output file
```bash
manx fastapi -o my_docs.md
manx fastapi -o results.json --json
```

## Environment Variables

### `MANX_API_KEY`
Set Context7 API key.

```bash
export MANX_API_KEY=sk-your-key
manx fastapi
```

### `MANX_CACHE_DIR`
Custom cache directory.

```bash
export MANX_CACHE_DIR=~/my-cache
manx react
```

### `MANX_DEBUG`
Enable debug mode.

```bash
export MANX_DEBUG=1
manx django
```

### `NO_COLOR`
Disable colored output.

```bash
export NO_COLOR=1
manx fastapi
```

## Advanced Usage

### Piping and Scripting

#### Extract specific content
```bash
manx fastapi --json | jq '.results[0].content'
```

#### Count results
```bash
manx react hooks --json | jq '.results | length'
```

#### Search and open in editor
```bash
manx fastapi --save 1 && vim manx_results_*.md
```

### Batch Operations

#### Search multiple libraries
```bash
for lib in fastapi django flask; do
  echo "Searching $lib..."
  manx $lib authentication --limit 3
done
```

#### Cache warming
```bash
# Pre-cache commonly used libraries
libs=("react" "vue" "angular" "fastapi" "django")
for lib in "${libs[@]}"; do
  manx $lib --limit 1 > /dev/null
  echo "Cached $lib"
done
```

### Integration Examples

#### Use with fzf for interactive selection
```bash
manx react hooks --json | \
  jq -r '.results[].title' | \
  fzf --preview "manx react hooks --json | jq -r '.results[] | select(.title==\"{}\")'" 
```

#### Create documentation file
```bash
#!/bin/bash
# save-docs.sh
libraries=("fastapi" "pydantic" "sqlalchemy")

for lib in "${libraries[@]}"; do
  echo "# $lib Documentation" >> project_docs.md
  manx doc $lib overview --quiet >> project_docs.md
  echo "" >> project_docs.md
done
```
