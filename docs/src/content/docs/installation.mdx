## System Requirements

- **Operating System**: Linux, macOS, or Windows
- **Architecture**: x86_64 or ARM64
- **Disk Space**: ~10MB (including cache)
- **Memory**: < 10MB RAM

## Installation Methods

<Tabs>
  <TabItem label="Cargo">
    ### Install via Cargo

    If you have Rust installed, this is the easiest method:

    ```bash
    cargo install manx-cli
    ```

    **Advantages:**
    - Always gets the latest version
    - Automatic updates with `cargo install --force manx-cli`
    - Compiled specifically for your system
  </TabItem>

  <TabItem label="Script">
    ### Automatic Installation Script

    Our installation script automatically detects your platform and installs the correct binary:

    ```bash
    # Using curl
    curl -fsSL https://raw.githubusercontent.com/neur0map/manx/main/install.sh | bash

    # Using wget
    wget -qO- https://raw.githubusercontent.com/neur0map/manx/main/install.sh | bash
    ```

    **What the script does:**
    1. Detects your operating system and architecture
    2. Downloads the appropriate binary
    3. Installs to `/usr/local/bin` (requires sudo)
    4. Verifies the installation
  </TabItem>

  <TabItem label="Manual">
    ### Manual Binary Installation

    1. **Download the binary for your platform:**

    | Platform | Architecture | Download |
    |----------|-------------|----------|
    | Linux | x86_64 | [Download](https://github.com/neur0map/manx/releases/latest/download/manx-x86_64-unknown-linux-gnu) |
    | Linux | ARM64 | [Download](https://github.com/neur0map/manx/releases/latest/download/manx-aarch64-unknown-linux-gnu) |
    | macOS | Intel | [Download](https://github.com/neur0map/manx/releases/latest/download/manx-x86_64-apple-darwin) |
    | macOS | Apple Silicon | [Download](https://github.com/neur0map/manx/releases/latest/download/manx-aarch64-apple-darwin) |
    | Windows | x86_64 | [Download](https://github.com/neur0map/manx/releases/latest/download/manx-x86_64-pc-windows-msvc.exe) |

    2. **Make executable (Unix-like systems):**
    ```bash
    chmod +x manx-*
    ```

    3. **Move to PATH:**
    ```bash
    # System-wide installation
    sudo mv manx-* /usr/local/bin/manx

    # User installation (no sudo)
    mkdir -p ~/.local/bin
    mv manx-* ~/.local/bin/manx
    echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </TabItem>

  <TabItem label="Source">
    ### Build from Source

    For developers who want the latest features:

    1. **Clone the repository:**
    ```bash
    git clone https://github.com/neur0map/manx.git
    cd manx
    ```

    2. **Build with Cargo:**
    ```bash
    cargo build --release
    ```

    3. **Install the binary:**
    ```bash
    sudo cp target/release/manx /usr/local/bin/
    ```

    **Development build:**
    ```bash
    cargo install --path .
    ```
  </TabItem>
</Tabs>

## Platform-Specific Instructions

### macOS

If you encounter "unidentified developer" warnings:

```bash
# Remove quarantine attribute
xattr -d com.apple.quarantine /usr/local/bin/manx
```

### Windows

1. Download the Windows executable
2. Add to PATH:
   - Open System Properties ‚Üí Environment Variables
   - Add the directory containing `manx.exe` to PATH
   - Restart your terminal

### Linux

Some distributions may require additional libraries:

```bash
# Ubuntu/Debian
sudo apt-get install libssl-dev

# Fedora/RHEL
sudo dnf install openssl-devel

# Arch
sudo pacman -S openssl
```

## Verification

After installation, verify everything is working:

```bash
# Check version
manx --version

# Run a test search
manx fastapi

# Check help
manx --help
```

## Updating Manx

### Via Cargo
```bash
cargo install --force manx-cli
```

### Via Script
```bash
curl -fsSL https://raw.githubusercontent.com/neur0map/manx/main/install.sh | bash
```

### Manual Update
1. Download the latest binary
2. Replace the existing binary
3. Verify with `manx --version`

## Uninstallation

### Remove Binary
```bash
# System installation
sudo rm /usr/local/bin/manx

# User installation
rm ~/.local/bin/manx
```

### Remove Configuration and Cache
```bash
rm -rf ~/.config/manx
rm -rf ~/.cache/manx
```

### Via Cargo
```bash
cargo uninstall manx-cli
```

## Troubleshooting

### Command not found
Make sure the installation directory is in your PATH:

```bash
echo $PATH
# Should include /usr/local/bin or ~/.local/bin
```

### Permission denied
```bash
# Fix permissions
chmod +x /usr/local/bin/manx
```

### SSL/TLS errors
Install or update your system's SSL certificates:

```bash
# macOS
brew install ca-certificates

# Linux
sudo update-ca-certificates
```
