## Quick Demo

```bash
# Search any library
manx fastapi

# Search with specific query
manx fastapi middleware

# Version-specific search
manx react@18 hooks

# Get comprehensive docs
manx doc fastapi authentication

# Export results
manx fastapi --save 1,3,5 --json
```
